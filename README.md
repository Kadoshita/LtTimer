# LtTimer

### C#で作ったLT用のタイマー

#### 5分だけ計れるLT用のタイマー

### 使用方法

* Assetsフォルダにstart.wav,end.wavというファイルを配置する。これらのファイルはカウント開始時、終了時に流れる音
* プロジェクトをVisualStudioで開き、実行(Windows8のストアアプリの開発、実行環境が必要)

### 動作環境

* Windows8.1
* Microsoft Visual Studio Community 2015 with Update 2
